# Recommended Books

Books are an excellent way to strengthen the theory behind penetration testing and cybersecurity. While the **eJPT** is primarily a practical certification, understanding the concepts behind the tools and techniques will make you a much better penetration tester.

This page contains some of the best books for beginners and intermediate learners, as well as recommended reading for those planning to continue toward certifications such as **eCPPT**, **PNPT**, **CRTO**, and **OSCP**.

---

# Beginner Books

## Penetration Testing: A Hands-On Introduction to Hacking

**Author:** Georgia Weidman

### Why Read It?

This is one of the best beginner books for penetration testing. It covers the complete penetration testing lifecycle while introducing many of the tools used during the eJPT.

### Topics Covered

- Linux Basics
- Networking
- Python Basics
- Nmap
- Metasploit
- Password Attacks
- Web Applications
- Wireless Security
- Exploitation

**Difficulty:** ⭐⭐☆☆☆

---

# Networking

## Computer Networking: A Top-Down Approach

**Authors:** James F. Kurose & Keith W. Ross

### Why Read It?

Understanding protocols makes enumeration significantly easier.

# Recommended Reading Order

If your goal is to pass the **eJPT** and continue toward the **OSCP**, the following order is recommended:

1. Linux Basics for Hackers
2. The Basics of Hacking and Penetration Testing
3. Penetration Testing: A Hands-On Introduction to Hacking
4. Computer Networking: A Top-Down Approach
5. The Web Application Hacker's Handbook
6. Black Hat Python
7. How Linux Works
8. Linux Privilege Escalation
9. Windows Privilege Escalation
10. Windows Internals

# Final Thoughts

Books alone will not make you a penetration tester. The best learning strategy is to combine reading with hands-on practice through platforms like **TryHackMe**, **Hack The Box**, **PortSwigger Web Security Academy**, and the **INE Penetration Testing Student** learning path.

For the **eJPT**, focus on mastering the fundamentals: networking, Linux, Windows, web applications, enumeration, exploitation, privilege escalation, and reporting. Once you are comfortable with these topics, you'll be well prepared not only for the certification but also for more advanced paths such as **eCPPT** and **OSCP**.
